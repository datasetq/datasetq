## [0.1.0] - 2024-10-14

### Added
- Initial release of dsq-core
- Core data processing functionality with jq-compatible syntax
- Support for multiple data formats (CSV, JSON, Parquet, etc.)
- DataFrame operations and transformations
- Filter compilation and execution engine
- High-level fluent API for data processing workflows
- Comprehensive error handling and type system

### Features
- Lazy evaluation for performance
- Columnar data processing with Polars
- Extensible operation framework
- Built-in functions registry
- Streaming and batch processing capabilities
